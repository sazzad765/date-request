# Date Request

A tiny romantic date invitation web app inspired by a playful "POV: you're dating an IT guy" video.

## Run Locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8766
```

Then visit `http://127.0.0.1:8766`.

## Deploy

This project is static and works on GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host.
